ℹ
No USB chip needed. The ESP32-S3 connects directly to the USB port via IO19 (D−) and IO20 (D+). You do not need a CH340, CP2102, or any other USB-to-UART bridge. Simply plug in a USB-C cable and your computer will enumerate the device. Drivers are built into modern operating systems.

05 — Getting Started

Quick Start Guide

From unboxing to running your first sketch in five steps. No drivers needed on modern systems.

ℹ
No USB chip on this board. If you're looking for a CH340 or CP2102 — there isn't one. The ESP32-S3 handles USB itself. Windows 10+, macOS, and Linux will enumerate the device automatically when you plug in the USB-C cable.
01
Install Arduino IDE 2 + ESP32 Board Package
In Arduino IDE 2, open File → Preferences and add the Espressif boards URL to Additional Boards Manager URLs:
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
Then go to Tools → Board Manager, search for esp32 and install the Espressif package.
02
Select the Correct Board
In Arduino IDE: Tools → Board → ESP32 Arduino → ESP32S3 Dev Module

Set the following options in the Tools menu:
// Recommended settings
Flash Mode → QIO 80MHz
Flash Size → 8MB (64Mb)
PSRAM → Disabled
USB Mode → USB-OTG (TinyUSB)
Upload Mode → UART0 / Hardware CDC
03
Enter Bootloader Mode (First Upload)
For the very first firmware upload, you may need to manually enter download mode:

1. Hold SW5 (BOOT) button down
2. Press and release SW6 (RESET / EN)
3. Release SW5

After the first upload, subsequent uploads can be done automatically over USB without pressing any buttons.
04
Blink the Built-in LED
Upload this sketch to verify everything is working:
#define LED_PIN 38 // GPIO38 = Builtin_LED (D12)

void setup() {
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_PIN, HIGH); delay(500);
  digitalWrite(LED_PIN, LOW); delay(500);
}
05
Read Battery Voltage
A 100k / 100k resistor divider on GPIO7 gives you half the battery voltage. Read it like this:
// Battery voltage is Vbat/2 on IO7
const int BAT_PIN = 7;

void setup() { Serial.begin(115200); }

void loop() {
  int raw = analogRead(BAT_PIN);
  float vbat = (raw / 4095.0) * 3.3 * 2.0;
  Serial.printf("Vbat: %.2f V\n", vbat);
  delay(1000);
}
Disconnect the divider via JP3 to eliminate the ~33 µA quiescent current during deep sleep.
